================================================================================
TROUBLESHOOTING QUICK FIXES
================================================================================

Problem: Database connection error
Fix: Check MySQL is running, verify database name is "mediarchive"

Problem: SOAP extension not found
Fix: Enable extension=soap in php.ini, restart Apache

Problem: QR code not generating
Fix: Check internet connection, verify qrcodes/ folder exists

Problem: Permission denied
Fix: Create uploads/, qrcodes/, temp/ folders with write permissions

Problem: Port 80 in use
Fix: Stop Skype or change Apache port to 8080
